# Spring Boot Arch Template
To explore and learn about software architecture, implementation, and testing methodologies in Spring Boot.

## Installation
- Git
- Java 21
- Docker

## Setup
```bash
$ git clone https://github.com/hiriorima/springboot-arch-template.git
```

## Start App
```bash
$ ./gradlew bootRun
```
